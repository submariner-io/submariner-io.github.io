Help Wanted

If you’d like to get involved and haven’t already found something to work on, check the GitHub Issues tagged “help wanted”.

Submariner’s success depends on growing the set of contributors to the project. Welcoming new contributors is a top priority of the project.

Research notice

Please note that this repository is participating in a study into sustainability of open source projects. Data will be gathered about this project’s GitHub repositories for approximately the next 12 months, starting from September 2021.

Data collected will include number of contributors, number of PRs, time taken to close/merge these PRs, and issues closed.

For more information, please visit the study informational page or download the participant information sheet.
